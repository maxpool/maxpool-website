Google

Foundational papers that shaped modern distributed systems

Google • 2003
Google File System (GFS)
Distributed file system for large-scale data processing. Learn how Google stores petabytes across thousands of commodity servers with automatic failure handling.
Storage Fault Tolerance Large Scale
Google • 2004
MapReduce Coming Soon
Simplified data processing on large clusters. The programming model that democratized big data processing.
Data Processing Distributed Computing
Google • 2006
BigTable Coming Soon
Distributed storage system for structured data. NoSQL database designed to scale to petabytes across thousands of machines.
NoSQL Storage
Google • 2012
Spanner Coming Soon
Globally-distributed database with external consistency. First system to distribute data at global scale with strong consistency.

Amazon

Dynamo Coming Soon
Highly available key-value store. Learn about eventual consistency, consistent hashing, and how to build for 99.99% availability.

Amazon • 2017
Amazon Aurora Coming Soon
Cloud-native relational database. Separation of compute and storage, with 6-way replication for durability.
SQL Cloud Native
